
# Routine Planner

App Flutter completo com:
- Telas: login, home, histórico, configurações, assinatura
- Cadastro de atividade com repetição semanal
- Compartilhamento com convites
- Controle de planos (grátis, individual, família)
- Simulação de Firestore
- Visual Material 3

Pronto para integração com Firebase real.
