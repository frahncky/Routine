# Play Graphics

Arquivos de apoio para os elementos graficos da loja.

## Feature Graphic

- `feature-graphic-routine-1024x500.svg`
- Tamanho: `1024 x 500`

## Como exportar para PNG

1. Abra o arquivo SVG no Figma, Canva, Photopea ou Inkscape.
2. Exporte em `PNG`.
3. Mantenha exatamente `1024 x 500`.
4. Envie no Play Console em:
   - `Store presence > Main store listing > Feature graphic`

## Dica rapida

Se quiser variar para teste A/B, duplique o SVG e altere apenas:
- texto de subtitulo
- cor do botao
- mockup do lado direito
